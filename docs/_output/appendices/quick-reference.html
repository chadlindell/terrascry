28  Quick Reference

28.1 Overview

This quick reference card provides essential information for field deployment of the HIRT system. Print this section as a laminated card for use in the field.

28.2 Power-Up Sequence

The HIRT quick setup sequence consists of seven steps:

  1. Connect Trunk Cables - Attach trunk cables from Zone Hubs to Central Hub
  2. Connect Probes - Connect probe cables to Zone Hubs
  3. Power ON Base Hub - Turn on the Central Hub power switch
  4. Wait 15s for Init - Allow system initialization to complete
  5. Verify “Ready” LED - Confirm the Ready LED is solid green
  6. Run System Diagnostic - Execute the built-in diagnostic test
  7. Start Measurement - Begin data acquisition
Note

Total Setup Time: approximately 2-3 minutes (excluding probe installation)
