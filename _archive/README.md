## Why Archived?

The project underwent reorganization to:
1. Consolidate the whitepaper as the single source of truth
2. Separate research from final documentation
3. Clean up the root directory
4. Establish clear project management files

## Retrieving Archived Content

If you need content from these files:
1. Check if it already exists in the whitepaper or hardware docs
2. If not, copy the relevant content to the appropriate location
3. Do not restore archived files to active directories
